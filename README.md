# 📊 Flipkart E-Commerce Sales Analysis

## 📌 Project Description
This project presents a comprehensive analysis of Flipkart’s e-commerce sales data using interactive dashboards and data visualization techniques. The goal is to derive meaningful insights related to sales performance, customer behavior, and operational efficiency.

The dataset used in this project has been sourced from **Kaggle**.

---

## 🔍 Overview
The dashboard highlights key business metrics and performance indicators, including:

- Total Sales Revenue
- Customer Footfall
- Average Bill per Customer
- Top Selling Products
- Category-wise Revenue
- Seller Performance
- Fulfillment Analysis
- Discount Impact on Sales
- Delivery Time Impact
- Payment Method Analysis
- Weekly & Monthly Trends
- City-wise Sales Performance

---

## 📈 Key Insights
- High-performing product categories contribute the majority of revenue.
- Discounts significantly influence purchase decisions.
- Faster delivery improves customer satisfaction and boosts sales.
- Certain sellers and cities dominate overall performance.
- Digital payment methods are the most preferred by customers.
